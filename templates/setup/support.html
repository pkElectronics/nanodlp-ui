IOT Control

By enabling the Cloud Connectivity feature you agree that anonymous ussage statistics are collected and processed by Concepts3D and it's affiliates

This includes information about completed prints, number of layers printed and duration of prints among others

The Cloud Service is a necessary requirement to use the printer in the Athena Mobile App

Therefore printer data is transmitted to servers operated by Concepts 3D in Europe

To make the data available in the app, following pieces of information are transmitted

  • Machine identifier and Serial Number
  • Machine status including resin profiles and usage statistics
  • Camera feed (if available)
  • Print jobs available on the machine (but no preview images or stl files)
  • Print progress and associated status messages

As we believe transparency about how and where you data is processed is important, following services are operated.

System / Service Function Provider / Platform Location / Region Data Stored / Processed Notes
olymp.concepts3d.eu Printer management, machine-to-machine authentication, update management Hetzner EU Central Printer serial numbers, user-printer mappings, user-device mappings, print job preview images optional, timelapses optional Central management endpoint
hades.concepts3d.eu MQTT server Hetzner EU Central Printer live data Central IoT MQTT broker
talos.concepts3d.eu User authentication and access management Auth0 US-4 User login credentials, social login tokens, user roles Central user management
hermes.concepts3d.eu Object storage Hetzner EU Central Athena software update binaries None
